# DevSphere AI

AI-powered futuristic coding platform for generating frontend projects, understanding code, and testing applications live.

---

# Features

- AI project generation using Gemini AI
- HTML CSS JavaScript generation
- Beginner-friendly explanations
- Monaco code editor
- Live preview playground
- Authentication system
- Save projects
- Prompt history
- Download generated files
- Responsive glassmorphism UI
- 3D futuristic design

---

# Tech Stack

## Frontend
- React
- Vite
- Tailwind CSS
- Framer Motion
- Monaco Editor

## Backend
- Node.js
- Express.js
- MongoDB
- Gemini AI API
- JWT Authentication
